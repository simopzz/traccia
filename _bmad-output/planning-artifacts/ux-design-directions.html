traccia Design Directions

Four variations within the Swiss/Brutalist direction, each exploring different answers to the open design questions: event card style, duration representation, timeline structure, and information density. All share the same color system (teal accent, signal palette), typography (Inter + JetBrains Mono), and spacious layout.


A: Swiss + Spine ‚Äî Bold bordered cards with hard shadows, connected by a vertical timeline spine. Tiered card heights for duration. Most "brutalist."

B: Accent Bar ‚Äî Minimal cards with a colored left border per event type. Time column on the left. Duration as a pill badge. Clean and scannable.

C: Fixed Height + Duration Bar ‚Äî All cards same height for consistency. Duration shown as a colored bar at the bottom. Compact, data-dense, Linear-inspired.

D: Notion Blocks ‚Äî Borderless cards with type tags and generous spacing. Tiered heights. Most spacious and calm. Notion-inspired.

Comparison

Side-by-Side Comparison

Key trade-offs between the four directions.

Card Style
A: Swiss Bordered ‚Äî Bold, distinctive, most "brand." Hard shadows add depth. Risk: can feel heavy with many events.

B: Accent Bar ‚Äî Light, scannable, type-color prominent. Risk: less distinctive, could feel generic.

C: Fixed Height ‚Äî Consistent rhythm, data-dense. Duration bar adds visual interest. Risk: all events look same size ‚Äî less "shape of the day."

D: Notion Blocks ‚Äî Spacious, calm, hover-reveals. Risk: feels less like a specialized tool, more like a generic block editor.
Duration Representation
A: Tiered heights ‚Äî Short/medium/long card padding. Subtle, natural. "Noticeable difference" without strict proportionality.

B: Duration pill ‚Äî Text badge ("2h"). Explicit, no ambiguity. Duration is read, not felt.

C: Duration bar ‚Äî Colored bar at card bottom. Visual + text. Adds scannable signal without changing card size.

D: Tiered heights + text ‚Äî Combines both. Most information, most spacious. Duration is both felt (height) and read (text).
Type Identification
A: Icon badge ‚Äî Colored square with emoji/icon. Strong visual anchor.

B: Left border color ‚Äî Fastest scanning. Color = type. But requires learning the mapping.

C: Letter dot ‚Äî Compact colored dot with letter (A/F/L/T). Space-efficient, explicit.

D: Type tag ‚Äî Text label with colored background ("Food", "Transit"). Most explicit, no learning curve. Takes more space.
Overall Feel
A: Field guide, authoritative, distinctive. The "portfolio piece" look.

B: Professional, clean, familiar. Would fit in any SaaS product.

C: Data-focused, efficient, compact. For users who want density.

D: Calm, spacious, modern. The "I just want to plan my trip" feel.
